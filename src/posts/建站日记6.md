## node自动关闭问题
每次我退出终端后，node就会自动关闭，就算我用了`nohup node build &`到后台任务，还是不行。
于是我去安装了`pm2`（注意，centos下光安装node还不行，还得单独安装npm，`sudo yum install npm`），然后使用`pm2 start build/index.js`开启服务就不会被随意关闭啦。
